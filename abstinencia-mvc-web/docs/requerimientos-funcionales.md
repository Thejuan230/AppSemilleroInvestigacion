# Requerimientos funcionales

- RF1: Registrar usuarios y autenticar sesiones.
- RF2: Permitir crear registro diario de abstinencia.
- RF3: Permitir registrar desencadenantes y estrategia de afrontamiento.
- RF4: Permitir registrar contactos de emergencia.
- RF5: Mostrar resumen estadistico de progreso.
- RF6: Mostrar historial reciente de registros.
