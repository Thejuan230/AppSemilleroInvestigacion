# Scrum Plan

## Roles
- Product Owner: define prioridades clinicas y funcionales.
- Scrum Master: facilita ceremonias y elimina bloqueos.
- Dev Team: implementa app y pruebas.

## Backlog priorizado
1. Autenticacion.
2. Registro diario.
3. Triggers y estrategias.
4. Contactos de emergencia.
5. Dashboard y reportes.
6. Pruebas y documentacion.

## Sprints
- Sprint 1: Base MVC + autenticacion.
- Sprint 2: Registro diario + validaciones.
- Sprint 3: Triggers + plan emergencia + dashboard.
- Sprint 4: Pruebas + UML + manuales.

## Ceremonias
- Daily: 15 min.
- Planning: inicio de sprint.
- Review: demostracion funcional.
- Retrospective: mejoras continuas.

## Definition of Done
- Codigo funcional probado.
- Requerimiento documentado.
- Evidencia en pruebas y manuales.
